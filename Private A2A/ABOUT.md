# About CypherPay

## Overview

CypherPay is a production-ready cryptocurrency payment protocol designed for Agent-to-Agent (A2A) communication systems. It provides a comprehensive framework for autonomous agents to conduct secure, transparent, and efficient on-chain transactions.
